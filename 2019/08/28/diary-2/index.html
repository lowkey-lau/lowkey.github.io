Markdown 写法记录

发表于 2019-08-28 分类于 干货

因为现在开始要记录博客了，而写博客的手法使用Markdown的写法，想要在这里记录一下收集到的写法，给自己作参考使用

一、关于标题层级

标题共有六个级别,使用 ( # )来区分层级

这是一级标题 （# 这是一级标题）

这是二级标题 （## 这是二级标题）

这是三级标题 （### 这是三级标题）

这是四级标题 （#### 这是四级标题）

这是五级标题 （##### 这是五级标题）
这是六级标题 （###### 这是六级标题）

二、关于字体设置

我是斜体
我是粗体
我是斜体加粗体
我是删除字符串
